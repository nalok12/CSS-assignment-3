Task 5:

Problem Statement

Explan the Z-index with a code example?

Solution:-

Z-index-This is the CSS property that controls the stacking oeder of overlapping element on a page. An element with a higher Z-index value will appear in front of an element with a lower Z-index value. The property is called Z-index because it sets the order of element along the Z-axis The Z_index property specifies the stack order of an element. An element with greater stack order is always in front of an element with a lower stack order.

Code Example:-If you have two elements with a Z-index of 0 and 1, the element witha Z-index of 1 will be in front of the element with a Z-index of 0.
